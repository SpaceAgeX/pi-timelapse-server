# Raspberry Pi Timelapse Server

## Hardware

- Raspberry Pi 4
- Logitech C922
- Elegoo Neptune 4 Max

## Stack

- FastAPI
- OpenCV
- FFmpeg
- HTML/CSS/JavaScript

## Features

- Live MJPEG preview
- Timelapse recording
- Automatic MP4 encoding
- Recording download
- Background camera reader

## Future Goals

- Background encoding
- Encoding progress
- Crash recovery
- Camera controls
- AI printer failure detection
